Lista não ordenada:

  • Páragrafo é escrito com o seletor p e ele é um seletor que possui text_content
    < p>esse é o text_content< /p>
  • A quebra de linha no html não é como em outros lugares, é necessário incluir o seletor < br />




    Veja que em tela temos várias linhas com ENTER.
  • Para criar uma linha horizontal utilizamos outro seletor que não tem text_content < hr />
  • Para mudar a fonte podemos usar o itálico < i>itálico< /i>
  • Para deixar a fonte em negrito < b>negrito< /b>
  • Para deixar a fonte sublinhado < u>sublinhado< /u>
